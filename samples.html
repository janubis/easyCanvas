Samples and demos

For best performance and stability we recommend the Chrome browser!

Basic paint using a minimal setup:

With just a few lines you can achieve paint capability with correctly handle mouse events.

See demo in new tab

Extended basic paint

This uses the same basis as the basic paint but is extended to use the built-in point and tool recorder to record strokes.

It also demonstrates redraw on resize, zooming, rotation, panning and download of the image:

Use mouse wheel to zoom or hold Alt key down and scroll wheel to rotate.

See demo in new tab

Grid demo

Demonstrates grid support that quantize the mouse positions as well provides functions to quantize other position data (updated for version 1.2).

See demo in new tab

Draw an image proportional to any destination rectangle

With the drawImageProp() method you can draw an image proportionally, filled and centered to any destination region.

The loadImage() method is also demonstrated here.

See drawImageProp demo in new tab here

Load multiple images

Shows how the loadImages method can be used including progress bar.

Canvas as image file drop zone

easyCanvas comes with a drop mode when active can be used to drop images directly on the canvas element.

The images are loaded and handed to your ondrop event ready to go.

See drop zone demo in new tab

Hit regions

Allows you to define hit regions with ID. Regions comes with events for mouse down, over and click.

Hit regions demo in new tab

Selection mode demo

Activates selection mode and draws a selection region when mouse is clicked and dragged on the canvas.

Get and set pixel demo

Demonstrates the getPixel method. To use setPixel hold mouse button down and move around. For the demo it uses only a white color.

See demo in new tab

Get and set pixel with bitmap locking demo

Demonstrates the effect of bitmap locking before using multiple getPixel and setPixel.

See demo in new tab

Velocity demo

Demonstrate how velocity can be used. Grab the ball and throw it and it will fly off in the direction you throw it with the velocity you throw it at.

This could also be a swipe move to animate elements.

See demo in new tab

Show detailed info from the extended event

See all details that are provided through the evens for angle, distance, velocity and more. Updated for version 1.2.

See demo in new tab

Fullscreen mode

Goes fullscreen in browsers which support it and handles the canvas offset, size and auto-clear buffer if in use.
