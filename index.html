What is easyCanvas for JavaScript?

easyCanvas is a free and powerful light-weight mini-framework that let you setup a HTML5 Canvas, context and properly handle mouse events, zoom, rotate, pan, image loading and background updates as well as providing features such as grid quantizing of positions, real-time calculated angles, velocity and distances, and give you features such as fullscreen modes and a select mode.

easyCanvas comes with a hotkey support, file drop capability, built-in points and tool recorder in addition to providing first point, previous point, time stamp and much more.

All available with a single line of code (saving you thousands of lines and many many hours!).

A stripped down version is also available as easyCanvasLT or easyCanvas light when you only need the core features such as setting up a canvas and context and correctly handle mouse events.

easyCanvas doesn't lock you in or limit you but allow you to use standard context and canvas operations without any restrictions. The main purpose with easyCanvas is to deal with the most tideous and repetitive tasks as well as assisting you with the more complex low-level stuff so you can focus on what you really want to make.

easyCanvas is good for games, design and paint applications and everywhere you need a quick setup of a canvas and context.

Setting up a powerful canvas environment has never been this easy!

How to use easyCanvas

To make a very basic paint application using easyCanvas or easyCanvasLT:

var ez = new easyCanvas();    // this sets up everything

ez.onmousemove = function(e) {
    ez.line(ez.prevX, ez.prevY, e.x, e.y);
}

Which could be written as this:

(new easyCanvas()).onmousemove = function(e) {
    this.line(this.prevX, this.prevY, e.x, e.y);
}

In this case easyCanvas does the following for you:

  • Obtains the context of the existing element
  • Handles mouse down events and sets mouse-is-down flag
  • Stores the point at mouse down
  • Handles mouse move event when mouse is down
  • Calculates the X and Y position relative to canvas including border and padding
  • Keeps track of previous point
  • Handles mouse up event
  • Does all calls necessary to draw the line

The equivalent code if you use vanilla setup would be something like this:

var canvas = document.getElementById('demo'),
    ctx = canvas.getContext('2d'),
    isMouseDown = false,
    prevX,
    prevY;
	
canvas.onmousedown = function(e) {

    e = e || window.event;

    var rect = canvas.getBoundingClientRect(),
        x = e.clientX - rect.left,
        y = e.clientY - rect.top;
	
    prevX = x;
    prevY = y;
	
    isMouseDown = true;
}

window.addEventListener('mousemove', function(e) {

    if (isMouseDown === true) {

        e = e || window.event;

        var rect = canvas.getBoundingClientRect(),
            x = e.clientX - rect.left,
            y = e.clientY - rect.top;
	
        ctx.beginPath();
        ctx.moveTo(prevX, prevY);
        ctx.lineTo(x, y);
        ctx.stroke();

        prevX = x;
        prevY = y;
    }
}, false);

window.addEventListener('mouseup', function(e) {
    isMouseDown = false
}, false);

That's a little more typing - and of course, easyCanvas do a bunch of more things such as distance, angle and velocity measure, panning, selection, zoom and rotation, file drop, image loading ++
