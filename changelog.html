Change log

Version 1.3

  • NEW: Added support for CORS mode for image loading (property useCrossOrigin and crossOriginMode)
  • NEW: Faster clear() in Chrome. Detects which method to use for clearing the canvas more efficiently.
  • FIX: Anti-alias was reset after a reset.

Version 1.2

  • NEW: Auto-Clear handles background updates when drawing (automatic and manual modes)
  • NEW: Drop mode for canvas - drop images into canvas. Event driven (ondrop, ondropstart, ondropprogress).
  • NEW: Fullscreen mode (in browsers which supports this) by a single method fullscreenMode (event onfullscreen).
  • NEW: Selection mode with support for custom selection shape. Event driven (onselectstart, onselect, onselectend)
  • NEW: Basic built-in hit region support with callbacks ( onregiondown, onregionover, onregion)
  • NEW: Fast pixel setting and getting with bitmap locking for multi-updates
  • NEW: Get grid cell indexes from position when using grid
  • NEW: Function to calculate grid cell indexes into position when using grid
  • NEW: Easily load multiple images from array with correct load handling and callback
  • NEW: Add hotkeys to easyCanvas with addHotkey(key, isAlt, isShift, isCtrl).
  • NEW: Custom header for recorded points
  • NEW: One-liner to render internally recorded points for tool 0 (lines)
  • NEW: Get bounding rectangle (getRect) between two sets of absolute positions corrected for negative values
  • NEW: New event onmousewheel for mouse wheel handling with unifier for cross-browser support.
  • NEW: Before and after events for the pan, zoom and rotation methods.
  • NEW: Added support to programmatically set pan (in addition to automatic handling)
  • NEW: Mouse event (move, up) extension firstDistance, distance between first and current point
  • NEW: Mouse event (move, up) extension firstAngle, angle between first point and current point
  • NEW: setDash to set dash pattern for lines cross-browser (supports only browser's native capability).
  • NEW: textSelect to turn off that annoying text selection that takes places sometimes in the browser while f.ex drawing.
  • ! CHANGE: event signature for onmousedown, onmousemove, onmouseup, onmousewheel, x and y moved to event (LT not changed)
  • ! CHANGE: changed callback signatures for loadImage. See documentation for details..
  • CHANGE: renamed strokeLine -> line (old still available through this version)
  • CHANGE: renamed getOffscreenCanvas -> getOffScreenCanvas (old still available through this version)
  • CHANGE: renamed event onmouseup's: totalAngle -> firstAngle (old still available through this version)
  • FIX: drawImageProp now also properly clips the image to destination size.
  • FIX: cssClass method now works.
  • FIX: padding method now works.

Version 1.1.1 Full (non-official release) and version 1.1 LT

  • HOTFIX: Fix for relative mouse positions in Firefox when using padding on the Canvas
  • HOTFIX: Fix for padding which accidentally used parameters keyword instead of arguments. (full only)

Version 1.1

  • NEW: Grid support with offset support and quantizing of mouse positions
  • NEW: Draw image proportionally to destination region
  • NEW: Correctly load an image with callback
  • NEW: Internal point / tool recorder with stroke arrays.
  • NEW: More complex velocity function that uses moving averages
  • NEW: Set and get pixels
  • NEW: strokeLine method (now line from version 1.2)
  • REMOVED: debug mode
